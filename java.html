Cara Membuat Segitiga Bintang Dengan Javascript

Pada artikel kali ini saya akan membahas bagaimana cara membuat segitiga bintang dengan menggunakan JavaScript dengan menggunakan W3School. Terdapat beberapa pola yang sudah saya buat yang bisa kalian coba sendiri.

Segitiga Bintang JavaScript

Output

Penjelasan dan analisa dari program diatas adalah :

  • Menggunakan identifier const agar nilainya tidak bisa dirubah.
  • Membuat parameter baris, lalu kita buat pola yang isinya string kosong yang nanti akan kita isi dengan pola bintang.
  • Loop yang pertama cetak baris, inisialisasi variabel i dengan nilai awal 0. Dengan syaratnya adalah variabel i lebih kecil dari baris maka program akan dijalankan dan akan di increament atau ditambah 1 sampai syarat terpenuhi.
  • Mencetak karakter br yang berfungsi untuk ganti baris dan disimpan kedalam variabel pola.
  • Cetak bintang dengan memasukkan nested looping, dengan variabel j nilai awal 0, dengan kondisi selama j lebih kecil sama dengan i, kita tambahkan 1 atau di increament.
  • Maka program akan dijalankan dan akan di increament sampai syarat terpenuhi dan mecetak pola dengan karakter * dan disimpan ke dalam variabel pola.


Segitiga Bintang Terbalik JavaScript

Output

Penjelasan dan analisa dari program diatas Secara konsep untuk program diatas dan program ini kurang lebih sama namun yang membedakan ialah pada operator yang digunakan, yakni :

  • Menggunakan identifier const agar nilainya tidak bisa dirubah.
  • Membuat parameter baris, lalu kita buat pola yang isinya string kosong yang nanti akan kita isi dengan pola bintang.
  • Loop yang pertama cetak baris, inisialisasi variabel i dengan nilai awal 0. Dan syaratnya adalah variabel i lebih kecil dari baris maka program akan dijalankan dan akan di increament atau ditambah 1 sampai syarat terpenuhi.
  • Mencetak karakter br yang berfungsi untuk ganti baris dan disimpan kedalam variabel pola.
  • Cetak bintang dengan memasukkan looping didalam looping atau nested loop, dengan variabel j nilai awal baris, kalau kita masukkan nilai 10 pada variabel baris maka, nilai awalnya adalah 10.
  • Kondisinya j lebih lebih besar dari i maka, kita decreament yang kemudian nilai j akan dikurangi 1 tiap iterasinya.
  • Nilai j yang kita masukkan adalah 10 maka, nilai j = 10 dan nilai i = 0, berarti 10 lebih besar dari 1 dan true pola bintang akan dicetak,kemudian, iterasi kedua nilai i berubah menjadi 1 dan kita mulai dari awal begitu seterusnya.
  • Maka program akan dijalankan dan akan di decreament sampai syarat terpenuhi dan mecetak pola dengan karakter * dan disimpan ke dalam variabel pola.


Bujur Sangkar Bintang JavaScript

Output

Penjelasan dan analisa dari program diatas adalah :

  • Menggunakan identifier const agar nilainya tidak bisa dirubah.
  • Membuat parameter baris, lalu kita buat pola yang isinya string kosong yang nanti akan kita isi dengan pola bintang.
  • Loop yang pertama cetak baris, inisialisasi variabel i dengan nilai awal 0. Dan syaratnya adalah variabel i lebih kecil atau sama dengan dari baris maka program akan dijalankan dan akan di increament atau ditambah 1 sampai syarat terpenuhi.
  • for yang pertama dengan variabel i untuk menampilkan baris sedangkan perulangan for dengan variabel j untuk menampilkan kolom.
  • Loop yang kedua kita buat sama karena kita akan membuat kolom 10x10.
  • for berniali i akan memproses perulangan didalamnya yaitu variabel j. Program akan membuat baris pertama apabila perulangan for pada variabel j selesai.
  • Jika looping pada variabel i dan variabel j bernilai true maka program akan melakukan print pada bintang pertama kolom 1 baris 1, karna masih bernilai true maka program akan terus berjalan dan melakukan print lagi untuk variabel j yang bernilai 2 dan seterusnya hingga syarat terpenuhi.
  • Maka jika input yang dimasukkan 10, 11 tidak lebih kecil sama dengan 10 menjadi false dan langsung melaksanakan program bawahnya atau membuat baris baru.
  • Mencetak karakter br yang berfungsi untuk ganti baris dan disimpan kedalam variabel pola.
  • Dengan demikian program akan dijalankan dan akan di increament sampai syarat terpenuhi dan mecetak pola dengan karakter * dan disimpan ke dalam variabel pola.
